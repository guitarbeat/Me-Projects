# Archive

This directory contains historical documentation and artifacts that are no longer actively used but preserved for reference.

## Contents

### migration/
Documentation from previous migration efforts, including:
- Dependency analysis reports
- Import path mapping documentation
- Integration summaries
- Migration reports

These files provide historical context about how the project evolved and can be useful for understanding past architectural decisions.

## Purpose

The archive serves to:
1. Keep the root directory clean and focused on active development
2. Preserve historical context and decision-making rationale
3. Provide reference material for future migrations or refactoring efforts
4. Maintain a complete project history without cluttering the active workspace
