# whonix-xfce-utm-configs

This folder stores VM configuration assets for running Whonix Xfce in UTM or importing the same setup elsewhere from OVF.

## Contents

- `Whonix-Gateway.utm/` - UTM bundle for the gateway VM
- `Whonix-Workstation.utm/` - UTM bundle for the workstation VM
- `Whonix-Xfce.ovf` - OVF export of the Whonix Xfce setup

## Usage Notes

- Open the `.utm` bundles directly in UTM when working on macOS.
- Use the `.ovf` file as the portable import/export artifact.
- These VM assets are intentional deliverables and should remain tracked.
