# chrome-ext

This folder is the repository wrapper for the Chrome bookmark organizer workspace.

The active application lives in `chrome-bookmarks-mcp/`, which contains the bookmark processing scripts, the web interface, and the main project documentation.

## Layout

- `chrome-bookmarks-mcp/` - main app and README
- `.github/`, `.Jules/`, `.enforcer/` - repository metadata and automation support
- `CONTRIBUTING.md` - contributor guidance for the workspace

## Start Here

1. Read [`chrome-bookmarks-mcp/README.md`](chrome-bookmarks-mcp/README.md).
2. Run commands from `chrome-bookmarks-mcp/` instead of this wrapper directory.
3. Ignore `server.log` and `server.pid`; they are runtime leftovers and are not part of the source tree.
